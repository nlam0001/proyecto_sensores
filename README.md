# proyecto_sensores
Proyecto red de sensores 

Este repositorio contiene el cuadro de mando (*dashboard*) interactivo desarrollado en Streamlit para visualizar la telemetría de los sensores almacenada en la base de datos MySQL.

## Requisitos Previos

Antes de ejecutar la aplicación, asegúrate de tener instalado Python 3 en tu sistema y acceso a la base de datos MySQL local (`proyecto_sensores`).

## Instalación de Dependencias

Abre una terminal en la carpeta del proyecto e instala las librerías necesarias ejecutando el siguiente comando:

```bash
pip install streamlit mysql-connector-python pandas numpy

## Pasos para Ejecutar el Dashboard

Abre la terminal o consola de comandos del sistema.

Navega hasta la carpeta exacta donde se encuentra el archivo del script (app.py).

Inicia la interfaz web ejecutando el siguiente comando:

```bash
streamlit run app.py
